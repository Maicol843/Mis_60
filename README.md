# 🎂 Invitación Digital - Mis 60 Años

Esta es una aplicación web interactiva y responsiva diseñada como invitación para el 60º aniversario de **Ricardo Robles**. El proyecto combina una estética festiva con funcionalidades prácticas para los invitados, optimizada para ser compartida a través de dispositivos móviles.

## 🚀 Demo
Puedes ver la invitación en vivo en el siguiente enlace:
[https://maicol843.github.io/Mis_60/](https://maicol843.github.io/Mis_60/)

## 🛠️ Tecnologías Utilizadas

* **HTML5**: Estructura semántica del contenido.
* **CSS3**: Estilos personalizados utilizando variables (`:root`), Flexbox para el diseño y animaciones de estrellas y fuegos artificiales.
* **JavaScript (Vanilla)**: Lógica para la interactividad del sitio.
* **Bootstrap 5**: Framework utilizado para garantizar que el sitio sea totalmente adaptable (responsive).
* **Google Fonts**: Implementación de tipografías elegantes como *Allison*, *Alegreya SC* y *Roboto*.

## ✨ Características Principales

* **Diseño Mobile-First**: Estructura optimizada para una visualización perfecta en smartphones.
* **Confirmación de Asistencia**: Botón con enlace directo para confirmar la presencia vía WhatsApp.
* **Ubicación Integrada**: Acceso directo a Google Maps para facilitar la llegada de los invitados.
* **Animaciones Dinámicas**: Fondo con estrellas titilantes y efectos visuales de celebración.
* **OpenGraph Optimizado**: Configuración de meta-etiquetas para que el enlace muestre una vista previa atractiva con imagen al ser compartido en redes sociales.

## 📂 Estructura del Proyecto

```text
├── index.html          # Página principal de la invitación
├── formulario.html     # Sección de datos adicionales/confirmación
├── script.js           # Lógica y funciones interactivas
├── style.css           # Estilos personalizados y animaciones
├── frente.png          # Imagen de portada para OpenGraph
└── img/                # Directorio de recursos gráficos (globos, copas, etc.)
```
